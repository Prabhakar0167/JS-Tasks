Learn the basics of JavaScript variables and data types in a simple and easy way. Understand how data is stored, reused, and identified in programming.

Variables

A variable is used to store data. It is like a container or box where we keep values. Every variable has a name and a value. We use variables to store and reuse data in programs.

What is a Variable?

A variable is a named container used to store data so it can be used later in a program.

Variable Declaration

In JavaScript, variables can be declared using var, let, and const.

Rules for Naming Variables

Variable names should start with a letter, underscore (_) or dollar sign ($), and should not use spaces.

Scope

Scope tells where a variable can be accessed or used in the program.

Reassignment

Reassignment means changing the value of a variable after it has been created.

Data Types

A data type tells what kind of data a variable is storing. It defines the type of value such as number, text, true/false, etc.

1. Primitive Data Types

Primitive data types are basic types in JavaScript.

  • Number
  • String
  • Boolean
  • Undefined
  • Null
  • BigInt
  • Symbol

2. Non-Primitive Data Types

Non-primitive data types are more complex and can store collections of values.

  • Object
  • Array
  • Function
  • Date
